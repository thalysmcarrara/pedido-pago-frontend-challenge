# Pedido Pago frontend challenge

Este projeto visa comprovar conhecimento em desenvolvimento frontend com next.js


O Projeto possui a seguinte estrutura:

```
├── __tests__
├── public
├── src
│  ├── components
│  ├── pages
│  │   └── index
│  │   └── cargos
│  │   └── employee
│  ├── services
│  │   └── api
│  ├── styles
│  │   └── global.tsx
│  │   └── roleDetail.ts
│  │   └── employeeDetail.ts
│  │   └── style.ts
│  ├── types
│  └── repository
│     └── StrangerThings.js
├── services
│  └── StrangerThings.js
├── package-lock.json
└── package.json
```
---

## Tecnologias utilizadas

  - react.js;
  - next.js
  - typescript
  - styled-components
  - axios
  - polished.js
  - react-testing-library
  - jest
